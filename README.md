# Sistema de Gestión de Restaurante

## Estudiante

Kelly Yolanda Alvarado Tanguila

## Descripción

Este proyecto fue desarrollado en Python utilizando Programación Orientada a Objetos y organización modular. Permite registrar productos y clientes de un restaurante, almacenarlos mediante listas y mostrar la información en consola.

## Estructura

```
restaurante_app/
│
├── modelos/
├── servicios/
└── main.py
```

## Tipos de datos utilizados

- str
- int
- float
- bool
- list

## Reflexión

El uso de identificadores descriptivos facilita la comprensión del código. La organización modular permite mantener el proyecto ordenado y las listas ayudan a administrar múltiples objetos de forma sencilla.
